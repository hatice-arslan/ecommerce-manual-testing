 ### TC_EC_001 – Valid Login

*Precondition:* User must be registered.

*Steps:*
1. Navigate to login page
2. Enter valid email and password
3. Click Login button

*Expected Result:*
User should be redirected to homepage.

*Status:* Pass
 ### TC_EC_002 – Add Product to Cart

*Steps:*
1. Select a product
2. Click Add to Cart

*Expected Result:*
Product should be added to cart.

*Status:* Pass
 ### TC_EC_003 – Complete Purchase

*Steps:*
1. Go to cart
2. Click Checkout
3. Enter valid payment details
4. Confirm payment

*Expected Result:*
Order should be completed successfully.

*Status:* Fail
